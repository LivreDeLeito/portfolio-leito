🌐 Site vitrine – HEUREUX EMSEMBLE

Développement d’un site web dynamique pour une association, permettant la publication d’actualités et le partage de photos.

🎯 Objectifs

  • Mettre en ligne les activités de l’association

  • Proposer un formulaire de contact

🛠 Technologies utilisées

  • PHP – Gestion du back-end
  • MySQL – Base de données
  • HTML / CSS – Interface utilisateur

💡 Compétences développées

  • Connexion base de données
  • Création de pages dynamiques
  • Gestion d’upload d’images
  • Organisation du code

🤖 Application IA – Reconnaissance d’animaux

Projet réalisé en stage pour créer une application web locale capable de reconnaître différents animaux à partir d’images.

🎯 Objectifs

  • Expérimenter l’apprentissage automatique en local
  • Entraîner une IA à identifier différents animaux
  • Développer une interface web pour visualiser les résultats
  • Tester et améliorer les modèles d’IA

🛠 Technologies utilisées

  • React – Développement de l’interface utilisateur
  • JavaScript / Node.js – Fonctionnalités interactives et traitement des données
  • Git / GitHub – Gestion du code et collaboration
  • Bibliothèques IA (TensorFlow.js ou équivalent) – Entraînement et détection d’images

💡 Compétences développées

  • Développement d’une application web expérimentale
  • Apprentissage automatique et traitement d’images
  • Intégration front-end avec React
  • Travail collaboratif via Git
  • Test et amélioration des performances de l’IA
